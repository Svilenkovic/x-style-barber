# X Style Barber

X Style je Next.js sajt sa 3D animacijama za premium barber brend, sa fokusom na atmosferu, animirani storytelling i moderni UX.

## Tehnologije

- Next.js (App Router)
- Tailwind CSS
- GSAP i Three.js
- TypeScript

## Lokalni razvoj

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
