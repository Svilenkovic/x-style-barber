# Architecture Overview

## Tip projekta

Static-export Next.js PWA deployovan preko Nginx-a.

## Operativni model

- Build pipeline generise staticke artefakte
- Artefakti se serviraju iz web root foldera
- Nginx obezbedjuje cache i security zaglavlja

## Napomena

Kompletan source nije ukljucen u javni repo.
Repo sluzi kao portfolio demo i infrastrukturna referenca.
